Firefox の次のリリースの最初の開発者マイルストーン（コードネーム名 Shiretoko Alpha 1）がダウンロード可能になりました。Shiretoko は Firefox のような リッチインターネットアプリケーションのコアを成す Gecko 1.9.1 プラットフォームのプレリリース版で作成されています。このリリースは開発者及びテスタに向けてのみを意図していることに注意してください。

この Shiretoko / Gecko 1.9.1 Alpha 版はいくつかの新機能を導入しています：

  • Gecko レイアウトエンジンにおける Web 標準の改善
  • <canvas>要素のための Text API
  • border 画像利用のサポート
  • JavaScript クエリセレクタのサポート
  • スマートロケーションバーのいくつかの改善
  • 新しいタブ切り替え動作

Shiretoko に興味があるならばダウンロードの前にリリースノートと Mozilla Developer Center の Firefox 3.1 For Developers（日本語訳）の記事を読むべきです。以下のリンクから Shiretoko をダウンロードしてください：

  • Windows: Shiretoko Alpha 1 Setup.exe
  • Mac OS X: Shiretoko Alpha 1.dmg
  • Linux: shiretoko-alpha1.tar.bz2
